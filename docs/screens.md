# 画面設計

## 画面一覧

| 画面ID | 画面名 | 概要 | 関連機能 |
|---|---|---|---|
| S-01 | ログイン画面 | メールアドレス・パスワードでログインする | F-02 |
| S-02 | サインアップ画面 | 新規アカウントを作成する | F-01 |
| S-03 | タイムライン画面 | フォロー中ユーザーの投稿を時系列表示。投稿作成の起点にもなる | F-03, F-04, F-05 |
| S-04 | 投稿詳細画面 | 投稿本文・画像・いいね数・コメント一覧(ネスト表示)を表示 | F-06, F-07, F-08, F-09, F-10, F-11 |
| S-05 | プロフィール画面 | 対象ユーザーの投稿一覧・自己紹介・フォロー数/フォロワー数、フォロー/フォロー解除ボタン | F-12, F-13 |
| S-06 | プロフィール編集画面 | 自分の表示名・自己紹介・アイコンを編集 | F-14 |
| S-07 | ユーザー一覧/検索画面 | フォロー対象ユーザーを探す | F-15 |
| S-08 | フォロー中/フォロワー一覧画面 | フォロー中・フォロワーのユーザー一覧 | F-18 |

## 画面遷移(概要)

```mermaid
graph TD
    S01[ログイン画面] -->|ログイン成功| S03[タイムライン画面]
    S01 -->|新規登録へ| S02[サインアップ画面]
    S02 -->|登録成功| S03

    S03 -->|投稿をタップ| S04[投稿詳細画面]
    S03 -->|ユーザー名をタップ| S05[プロフィール画面]
    S03 -->|検索/一覧| S07[ユーザー一覧/検索画面]

    S04 -->|投稿者名をタップ| S05
    S04 -->|コメント投稿者名をタップ| S05

    S05 -->|フォロー数/フォロワー数をタップ| S08[フォロー中/フォロワー一覧画面]
    S05 -->|編集ボタン(自分のみ)| S06[プロフィール編集画面]
    S05 -->|投稿をタップ| S04

    S07 -->|ユーザーをタップ| S05
    S08 -->|ユーザーをタップ| S05
```

## 各画面の要素

### S-03 タイムライン画面
- ヘッダー: ロゴ、プロフィールアイコン(自分)、ログアウト
- 投稿作成エリア: テキスト入力欄、画像添付ボタン、投稿ボタン
- 投稿カード(リスト): 投稿者アイコン・表示名、投稿本文、添付画像、いいね数、コメント数、いいねボタン
- ページネーション/無限スクロール

### S-04 投稿詳細画面
- 投稿本文・添付画像・投稿日時
- いいねボタン、いいね数
- コメント一覧(ネスト表示。返信は親コメントの下にインデントして表示)
- コメント入力欄(投稿への直接コメント、または特定コメントへの返信)

### S-05 プロフィール画面
- アイコン、表示名、自己紹介
- フォロー数、フォロワー数(タップでS-08へ)
- フォロー/フォロー解除ボタン(他人のプロフィールの場合のみ)
- 本人の投稿一覧

### S-07 ユーザー一覧/検索画面
- 検索ボックス(表示名検索)
- ユーザーリスト(アイコン・表示名・フォローボタン)
