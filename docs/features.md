# 機能一覧

## 凡例

- 優先度: Must(初期リリース必須) / Should(初期リリースで対応したいが調整可) / Could(将来拡張)

| No | 機能名 | 概要 | 優先度 |
|---|---|---|---|
| F-01 | ユーザー登録(サインアップ) | メールアドレス・パスワード・表示名でアカウントを作成する | Must |
| F-02 | ログイン/ログアウト | メールアドレス・パスワードで認証し、セッションを開始/終了する | Must |
| F-03 | タイムライン表示 | フォローしているユーザーの投稿を新着順に一覧表示する | Must |
| F-04 | 投稿作成 | テキスト(+画像)で投稿を作成する | Must |
| F-05 | 画像投稿 | 投稿に画像を1枚以上添付する | Must |
| F-06 | 投稿詳細表示 | 投稿本文・添付画像・コメント一覧・いいね数を表示する | Must |
| F-07 | コメント投稿 | 投稿に対してコメントを行う | Must |
| F-08 | コメント返信(ネスト) | 既存のコメントに対して返信を行う | Must |
| F-09 | コメント数表示 | 投稿ごとに、返信を含むコメント総数を表示する | Must |
| F-10 | いいね/いいね解除 | 投稿に対していいねを行う、または取り消す | Must |
| F-11 | いいね数表示 | 投稿ごとに、いいねをした人数を表示する | Must |
| F-12 | フォロー/フォロー解除 | 他ユーザーをフォロー、またはフォロー解除する | Must |
| F-13 | プロフィール表示 | ユーザーの表示名・自己紹介・投稿一覧・フォロー数/フォロワー数を表示する | Must |
| F-14 | プロフィール編集 | 自分の表示名・自己紹介・アイコン画像を編集する | Should |
| F-15 | ユーザー一覧/検索 | フォロー対象を探すためのユーザー一覧・検索 | Should |
| F-16 | 投稿削除 | 自分の投稿を削除する(論理削除) | Should |
| F-17 | コメント削除 | 自分のコメントを削除する(論理削除) | Should |
| F-18 | フォロー中/フォロワー一覧 | 自分・他ユーザーのフォロー中/フォロワーの一覧を表示する | Could |
| F-19 | 通知機能 | いいね・コメント・フォローされた際の通知 | Could |
| F-20 | 投稿検索・ハッシュタグ | キーワードやハッシュタグでの投稿検索 | Could |
